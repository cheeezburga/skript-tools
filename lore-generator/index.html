Guide

Color Codes: Use &0 to &f for colors.

Formatting Codes: &l for bold, &o for italic, &n for underline, &m for strikethrough, &k for obfuscated.

Reset Code: &r to reset formatting.

Alternate Color and Formatting: Use <color>, <format>, or <#hexcode> for more options.

Default Formatting: Without any codes, the lore text is purple and italicized.

Special Characters: To include a double quote ", double it like "".
